# Production Site Crawler

A production-oriented web crawler take-home assignment built around a TypeScript/Node.js CLI, PostgreSQL-backed crawl state, and an external Fetch API. The implementation will crawl from a seed URL, persist fetched content by type with metadata, and support durable, resumable, concurrency-safe execution in later phases.
